# Semana 11 - AgroData Integridad Pro

Proyecto académico para trabajar el tema:

**Manejo de archivos y datos simples; integridad y organización de información.**

Este ZIP contiene dos espacios:

```text
carpeta_estudiante/   Material que se entrega a estudiantes.
carpeta_profesor/     Guía, solución final y criterios de evaluación.
```

## Enfoque didáctico

El proyecto simula un sistema web para revisar datos rurales almacenados en archivos `.csv` y `.json`. La aplicación usa librerías externas instaladas con `npm` y copiadas a una carpeta local `vendor/`. Si las librerías no están instaladas, la página carga con errores visibles para que los estudiantes aprendan a diagnosticar dependencias faltantes desde VS Code, consola del navegador y pestaña Network.

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript modular
- CSV
- JSON
- Markdown
- Node.js / npm para preparar librerías
- PapaParse
- Chart.js
- SweetAlert2
- Font Awesome
- jsPDF
